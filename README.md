# chat-app-mern


App Chat Real Time 

Front-End: ReactJS, Chakra-ui

Back-End: NodeJS, JWT, Socket, MongoDB

Function: 
 + Sign In, Sign Up (Upload Avatar), Log out
 + View Profile
 + Chat 1-1 (User-User) 
 + Chat 1-n (Group Chat)
 + Typing Message
 + Notification new message
 + Delete User in Group (Only Admin)
 + Update Group Chat
 + Search Member
 
 
 Deploy : https://chat-app-ttp.netlify.app/
